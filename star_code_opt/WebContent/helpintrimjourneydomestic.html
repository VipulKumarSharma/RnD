The Domestic Intermediate Destinations Page allows a User/Administrator to add intermediate destinations.
Domestic Intermediate Destinations
Intermediate Destinations

This page allows a User to add Domestic intermediate destinations.The user is required to provide the following information.

  1. Departure City- It is a mandatory field and cannot be left blank.User needs to enter the intermediate departure city of the journey.

  2. Arrival City- It is a mandatory field and cannot be left blank.User needs to enter the intermediate arrival city of the journey.

  3. Departure Date- It is a mandatory field and cannot be left blank.User needs to select the departure date of the journey from the STARS date picker.

  4. Mode - It is a mandatory field and the user has to select the mode of transport that the traveller will take during departure/Arrival from the list, by default its value is set to "Air"

  5. Name of Airline/Train- It is a mandatory field and can not be left blank.Request the user for the name of the Airline/Train whichever mode of transport is preferred for the journey.

  6. Class - It is a mandatory field and the user needs to select the class of the mode of transport from the list.by default its value is set to "Eco"

Intermediate Destinations Details
  1. Intermediate destination Details -All the domestic intermediate destinations details entered by the user will be shown in the table "Intermediate destination Details"
Control Buttons
  1. Add - This button will add the entered intermediate journey details in the table Intermediate Destinations details.It is necessary to fill all the above mentioned mandatory information before adding them.

  2. Close - This button will close the Domestic intermediate destination page and come back on the Domestic Travel request page.

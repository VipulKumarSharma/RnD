This page allows a User to Login on STARS. The user is required to enter the following information:

  1. Username - To login on STARS, the user has to enter his/her Username which was entered at the time of registration, in the Username field .This field is Case sensitive.

  2. Password - To login on STARS, the user has to enter his/her Password which was entered at the time of registration,in the Password field .This field is Case sensitive.
New Registration
  1. New registration - It allows the New User to register himself in STARS. The user is required to enter the necessary information in the NEW User Registration Page which will appear on clicking the link "New Registration". After registration user will be activated only by Local Administrator.
Help Desk
  1. Helpdesk - It allows the user to contact the administrator through mail in case of any problem or to ask any query.

The Create Domestic Travel requisition Page allows a User/Administrator to perform the following activities:
Create Domestic Travel Request
Basic Information
Personal Information

This page allows a User to create an Domestic Travel Request on the STARS.The user is required to provide the following information.

  1. Select your Unit - It is a mandatory field and cannot be left blank. User needs to choose a given unit of the Traveller from the list. Only one company can be chosen;there is no option for multiple selection.

  2. Select your Name - It is a mandatory field and cannot be left blank.The user needs to choose traveller's name from the list.

  3. Designation - It is a non editable field.This field shows the designation of the traveller.It will be automatically filled when the user select the traveller's name;

  4. Age - It is a non editable field.This field shows the age of the traveller.It will be automatically filled when the user select the traveller's name;

  5. Contact number - It is a mandatory field and can not be left blank.This field request the user to fill the contact number.

  6. Frequent Flyer Number -It is a mandatory field and can not be left blank.The user needs to Specify the Frequent Flyer Number of the traveller.

  7. Sex - It is a mandatory field,by default the vlaue is set to "Male " but the user can select the any of the radio button either " male" or "female " accordingly

  8. Meal Preference- It is a mandatory field.By default its value is set to "VEG" but the User can choose his/her mail preference from the list.

  9. Departure City- It is a mandatory field and cannot be left blank.User needs to enter the departure city of the journey.

  10. Arrival City- It is a mandatory field and cannot be left blank.User needs to enter the arrival city of the journey.

 
Add Itinerary
  1. Departure Date- It is a mandatory field and cannot be left blank.User needs to select the departure date of the journey from the STARS date picker.

  2. Return Date- It is not a mandatory field and can be left blank.Request the User to select the Return date of the journey from the STARS date picker.

  3. Preferred time- It is a mandatory field and the user needs to select its preferred time for departure/arrival.
    By default its value is set to "Anytime" but the User can choose any value from the list.

  4. Mode - It is a mandatory field and the user has to select the mode of transport that the traveller will take during departure/Arrival from the list, by default its value is set to "Air"

  5. Preferred Airline/Train- It is a mandatory field and can not be left blank.Request the user for the name of the Airline/Train whichever mode of transport is preferred for the journey.

  6. Class - It is a mandatory field and the user needs to select the class of the mode of transport from the list.by default its value is set to "Eco"

  7. Intermediate destinations - Press this button if the traveller will be taking intrim journey for including the intermediate destinations.

Credit Card Information
  1. Card Type -It is not mandatory field ,its default value is set to N/A(Not applicable).If the Traveller use Credit card,then the user can select the credit card type from the list.It can be "Visa" Or "Master"

  2. Credit Card -It is not a mandatory field,but if the value of card type field is other than N/A then it can't be left blank.Request the user to input the credit card number.

  3. Expiry Date - It is not a mandatory field,but if the value of card type field is other than N/A then it can't be left blank.The user needs to input the expiry date of the credit card.

[ TOP ] 
Accommodation Required
  1. Transit Type - If the Traveller require Hotel Booking then choose the desired Transit type from the list and mention the budget and currency.If it is not required then choose N/A

  2. Currency - If the Traveller require Hotel/Transit house then choose the currency in which the budget is required from the list

  3. Budget - If the Traveller require Hotel/Transit house then specify the Budget.

  4. Check In Date - If the Traveller require Hotel/Transit house then pick the check-in date fronm the STARS date picker

  5. Check Out Date - If the Traveller require Hotel/Transit house then pick the check-out date fronm the STARS date picker

  6. Remark for other services - If the Traveller require any other services ,then specify the remarks in this text field.

[ TOP ] 

Control Buttons
  1. Save - This button will only save all the entered information.It is necessary to fill all the mandatory information before saving them.

  2. Save & Next - This button will save all the entered information and proceed to the next page.It is necessary to fill all the mandatory information before proceeding to the next page.

  3. Save & Exit - This button will save all the entered information and will Exit.It is necessary to fill all the mandatory information before save and Exit.

  4. Back - This button will take the user to the previous page.

Billing Instructionss







Expenditure Head

















 
  1. Chargeable to - The user can choose any of the following options.
    • Self- User can select this option when the bills are chargeable to the company to which originator(the person who is originating the request) belongs to.
    • SMG-The User can choose this option when the bills are chargeable to the SMG company and then can select the company name from the list
    • Outside SMG-The User can choose this option when the bills are chargeable to company Outside the SMG.If Outside SMG then it is mandatory to enter the name of the client

  2. Travel Allowances - It is a mandatory field.The User has to enter Expenditure/Day and total No.of tour days,the total amount required will be automatically generated for the travel Allowances.

  3. Daily Charges - It is a mandatory field.The User has to enter Expenditure/Day and total No.of tour days,the total Amount required will be automatically generated for Daily charges.

  4. Contigencies - It is not a mandatory field.The User can enter the amount for Contigencies and it is also mandatory to specify the reason for the contigencies in the remarks

  5. Any Other expenditure - It is not a mandatory field.The User can enter the amount for Any Other expenditure and it is also mandatory to specify the reason for Any Other expenditure in the remarks

  6. Total - After entering all the above mentioned fields,the grand total will be generated automatically

  7. Attach File -Through Attach File option the user can upload files with .doc, .xls, .ppt, .gif, .txt, .jpeg, .html, .zip extension only.The document user is uploading will be written to the STARS Server and can be referenced by you anytime by clicking the attachment icon which will be present with the requisition itself.

Control Button Submit to Workflow -This button will save all the entered information and submit the requisition into the workflow for approval.It is necessary to fill all the mandatory information before submitting to workflow for approval.
